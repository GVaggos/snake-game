# 🐍 Snake Game (Python / Pygame)

A modern Snake game built with Python and Pygame, featuring smooth gameplay, sound effects, and dynamic difficulty.

---

## 🎮 Features
- 🖱️ Mouse-based menu
- ⚡ Dynamic difficulty (Easy → Impossible)
- 🔊 Sound effects & background music
- 🏆 Highscore system
- 🎨 Modern UI with rounded grid design

---

## 🕹️ Controls
- ⬆️⬇️⬅️➡️ Arrow keys → Move
- 🖱️ Mouse → Menu navigation
- 🔁 R → Restart
- ⎋ ESC → Back to menu

---

## 🛠️ Technologies
- Python 3
- Pygame

---

## 🚀 Run the Game


pip install pygame
python main.py
